# Data Dictionary Simulator & SAS Analytics Environment

## Visão Geral
Projeto desenvolvido para simular um ambiente analítico completo, unindo modelagem de dados,
documentação técnica, consultas SQL e análises estatísticas em SAS Enterprise Guide.

O objetivo é praticar análise de dados em um contexto próximo ao corporativo, com foco em
entendimento de regras de negócio, estatística aplicada e organização analítica.

---

## Estrutura do Projeto
- `/data-dictionary` → documentação de tabelas, colunas e regras analíticas
- `/sql` → consultas SQL (SAS SQL e T-SQL)
- `/sas` → scripts prontos para execução no SAS Enterprise Guide
- `/python` → simuladores e apoio à documentação
- `/html` → templates de visualização e documentação

---

## Data Dictionary
Simula um banco de dados relacional de seguros, contendo:
- definição de tabelas
- tipos de dados
- regras de negócio
- sensibilidade LGPD
- exemplos de valores

O data dictionary serve como base para todas as análises estatísticas e consultas SQL.

---

## Análises Estatísticas Implementadas (SAS)
O script SAS executa análises como:

- Estatística descritiva
- Segmentação de clientes
- Análises multivariadas
- Modelos preditivos
- Séries temporais
- Análise de sobrevivência
- Análises bayesianas
- RFM e churn

---

## Arquivo Principal SAS
